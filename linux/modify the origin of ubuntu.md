
## modify the origin of ubuntu

Modify domestic sources and improve the download speed of software updates

`cd /etc/apt`
`sudo cp source.list source.list.bak` //Back up source.list file
`sudo gedit source.list` //open source.list file with gedit
Replace everything in source.list file with the following

Website available in China (choose one)

